---
title: "Get Bin"
openapi: "GET /api/bins/{binId}"
description: "Retrieve details for a specific bin"
---

<ParamField path="binId" type="string" required>
  The bin's public ID or internal ID
</ParamField>

<ParamField header="x-bin-password" type="string">
  Password for protected bins
</ParamField>

## Request

<CodeGroup>
```bash cURL
curl https://curlme.io/api/bins/abc123xyz \
  -H "x-api-key: cm_your_api_key"
```

```bash cURL (protected bin)
curl https://curlme.io/api/bins/abc123xyz \
  -H "x-api-key: cm_your_api_key" \
  -H "x-bin-password: secret123"
```

```javascript JavaScript
const response = await fetch('https://curlme.io/api/bins/abc123xyz', {
  headers: { 'x-api-key': 'cm_your_api_key' }
});
const bin = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://curlme.io/api/bins/abc123xyz',
    headers={'x-api-key': 'cm_your_api_key'}
)
bin = response.json()
```
</CodeGroup>

## Errors

| Status | Description |
| ------ | ----------- |
| `401` | Password required for protected bin |
| `403` | Permission denied (not your bin) |
| `404` | Bin not found |
